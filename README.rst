===============
Yukon Solitaire
===============

Install
================

.. code-block::

   git clone https://github.com/wallrj/yukon.git
   virtualenv --python python3 /tmp/yukon
   /tmp/yukon/bin/pip install ./yukon


Play
====

.. code-block::

   /tmp/yukon/bin/yukon
